## How to use it
When you start the program, you'll see a simple menu. Just type the number for what you want to do:
- Type **1** to deposit money
- Type **2** to withdraw money  
- Type **3** to check your balance
- Type **0** when you're done

A few things to remember:
- You start with ‚Çπ10,000 in your account
- You can only do 3 transactions at a time
- Amounts must be in multiples of 100 (like real ATMs!)
- Maximum withdrawal is ‚Çπ20,000
